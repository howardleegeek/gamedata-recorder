&#x20; - WM\_DISPLAYCHANGE（显示器插拔）

&#x20; - WM\_DEVICECHANGE（设备变化）

&#x20; - 游戏从窗口切换到全屏（或反过来）

&#x20; - 游戏窗口移动到另一块屏幕





PS：

给 agent 的补充说明：



Game Capture 对应 OBS source type: game\_capture

Window Capture 对应 OBS source type: window\_capture

Display Capture 对应 OBS source type: monitor\_capture，需传入 monitor\_id 参数指定屏幕

显存查询用 IDXGIAdapter3::QueryVideoMemoryInfo

游戏全屏检测用 GetWindowRect 对比 GetMonitorInfo 的 rcMonitor 是否完全重合

游戏窗口轮询间隔建议 500ms
